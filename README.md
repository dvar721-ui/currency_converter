# 💱 Currency Converter - Конвертер валют

**Автор:** Юлиана Гришина

---

## 📖 Описание проекта

**Currency Converter** — это графическое приложение для конвертации валют с использованием актуальных курсов из внешнего API. Приложение позволяет конвертировать суммы между различными валютами, сохраняет историю конвертаций и предоставляет удобный интерфейс для работы.

---

## ✨ Функциональные возможности

### Основные функции
- Конвертация между 24 валютами
- Получение актуальных курсов из API
- Сохранение истории конвертаций в JSON
- Загрузка истории при запуске
- Очистка истории и полей ввода

## 🔑 Как получить API ключ

Для работы приложения необходим бесплатный API ключ от ExchangeRate API:

1. Перейдите на сайт: https://app.exchangerate-api.com/sign-up
2. Зарегистрируйтесь (можно через Google или email)
3. После регистрации вы получите бесплатный API ключ
4. В коде замените `YOUR_API_KEY_HERE` на ваш ключ:

```python
self.api_key = "ваш_ключ_здесь"
